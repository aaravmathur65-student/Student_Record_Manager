# Student Record Manager

A Java-based web application for managing student records using JSP, Servlets, and H2 Database.

## Features

* Login system
* Student management
* Course management
* Attendance tracking
* Result management

## Technologies Used

* Java (JSP + Servlet)
* JDBC
* H2 Database
* Apache Tomcat

## How to Run

1. Import project into Eclipse IDE
2. Configure Apache Tomcat (v9)
3. Right-click project → Run on Server
4. Open browser:
   http://localhost:8080/StudentRecordManager/layout

## Database

* H2 database included in `/data` folder
* No external setup required
